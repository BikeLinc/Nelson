# Nelson

Nelson is a C++ based 2D game engine and editor that is in very early stages of development.

## Building

### Clone repository
```bash
git clone https://github.com/bikelinc/nelson
```
### Navigate into build directory
```bash
cd build
```
### Run CMake
```bash
cmake ..
```
or
```bash
cmake -G "Visual Studio" ..
```
